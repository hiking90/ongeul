모드 인디케이터

한/영 전환 시 커서 근처에 현재 모드를 표시하는 인디케이터가 나타납니다.

표시 방식

  • 한글 모드: 강조 색상 배경에 한 표시
  • 영문 모드: A 표시
  • 1.6초 후 자동으로 사라집니다.

위치

인디케이터는 현재 텍스트 커서 위치 근처에 표시됩니다. 앱이 커서 위치 정보를 제공하지 않는 경우에는 화면 중앙 하단에 표시됩니다.
