Ongeul을 사용하려면 다음 단계를 진행하세요:

1. 로그아웃/로그인

macOS가 새 입력기를 인식하려면 로그아웃 후 재로그인이 필요합니다.

2. 입력 소스 추가

  1. 시스템 설정 → 키보드 → 입력 소스 → 편집...
  2. 좌측 하단의 + 버튼 → Ongeul 추가

3. 기존 입력 소스 제거 (권장)

Ongeul이 한글과 영문을 모두 처리하므로, ABC 등 다른 입력 소스를 제거하면 전환 지연이 사라집니다.

자세한 사용법은 온라인 문서를 참고하세요.
