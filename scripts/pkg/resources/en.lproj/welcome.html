Ongeul Korean Input Method

A macOS Korean input method that handles both Korean and English within a single input source.

Key Features

  • Single Input Source — Switch between Korean and English without changing input sources
  • Keyboard Layouts — 2-Set Standard, 3-Set 390, 3-Set Final
  • Mode Indicator — Shows current mode near the cursor
  • Per-App Mode Memory — Automatically restores the last input mode for each app
