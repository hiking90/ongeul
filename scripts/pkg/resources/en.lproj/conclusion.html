Installation Complete

To start using Ongeul, follow these steps:

1. Log Out / Log In

macOS requires a logout and re-login to recognize the new input method.

2. Add Input Source

  1. System Settings → Keyboard → Input Sources → Edit...
  2. Click + at the bottom left → Add Ongeul

3. Remove Other Input Sources (Recommended)

Since Ongeul handles both Korean and English, removing other input sources like ABC eliminates switching delays.

For detailed usage, see the online documentation.
